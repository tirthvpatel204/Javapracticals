# ☕ Java OOP Practical Programs

This repository contains Java practical programs demonstrating the concepts of **Object-Oriented Programming (OOP)** in Java.

---

## 📂 Programs Included

### 1️⃣ Thermostat Management System

**File:** `Thermostat.java`

### Description
A Java console application that simulates a thermostat for different rooms.

### Features
- Set default or custom temperature
- Increase and decrease temperature
- Minimum temperature limit: **16°C**
- Maximum temperature limit: **30°C**
- Count active thermostat objects using a static variable

### OOP Concepts Used
- Classes & Objects
- Constructors
- Constructor Overloading
- Static Variables
- Final Variables
- Methods

---

### 2️⃣ CinemaShow Management System

**File:** `CinemaShow.java`

### Description
A Java console application for managing cinema seat bookings.

### Features
- Create a cinema show with movie title and capacity
- Book seats
- Cancel booked seats
- Prevent overbooking
- Track total booked seats
- User input using `Scanner`

### OOP Concepts Used
- Classes & Objects
- Constructor Overloading
- Constructor Chaining (`this()`)
- Static Variables
- Final Variables
- Methods
- Scanner Class
