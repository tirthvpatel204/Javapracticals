# ☕ Java OOP Practical Programs

This repository contains Java console applications developed to practice Object-Oriented Programming (OOP), Java Enums, Records, Loops, Conditional Statements, and User Input.

---

## 📂 Programs Included

### 1️⃣ Rock Paper Scissors Lizard Spock (RPSLS)

**File:** `RPSLS.java`

### Description
A Java console game where the player competes against the computer in the extended version of Rock-Paper-Scissors.

### Features
- Five possible moves:
  - Rock
  - Paper
  - Scissors
  - Lizard
  - Spock
- Random computer move generation
- Five rounds of gameplay
- Score tracking
- Overall winner declaration

### Concepts Used
- Enum
- Switch Statement
- Random Class
- Scanner Class
- Loops
- Methods

---

### 2️⃣ Toll Booth Management System

**File:** `TollBooth.java`

### Description
A Java console application that simulates a toll booth by collecting toll charges based on vehicle type.

### Features
- Accept vehicle number and type
- Supports Bike, Car, and Truck
- Calculates total toll collected
- Displays the most frequent vehicle type
- Uses Java Record for vehicle details

### Concepts Used
- Java Record
- Switch Statement
- Loops
- Scanner Class
- Conditional Statements

---

### 3️⃣ Vending Machine Simulation

**File:** `VendingMachine.java`

### Description
A Java console application that simulates a vending machine accepting coins until the snack price is reached.

### Features
- Accepts coins:
  - ONE
  - TWO
  - FIVE
  - TEN
- Calculates total amount inserted
- Dispenses snack
- Returns change

### Concepts Used
- Enum
- Switch Statement
- Loops
- Scanner Class
- Conditional Statements
